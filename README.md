#  Customer Segmentation System

This project demonstrates **customer segmentation** using **K-Means clustering** on a simple dataset with `age` and `income`.  
It is designed to show how clustering can be used for **marketing insights**, **analytics**, and **customer profiling**.

---

##  GitHub Description
> Customer segmentation using clustering for analytics and marketing insights.

---


##  How It Works

1. Load the customer data (age and income).  
2. Apply K-Means clustering to segment customers into groups.  
3. Assign a `segment` label to each customer.  
4. Output the dataset with assigned segments.

---
How to Run


Install dependencies:

pip install -r requirements.txt


Run the script:

python main.py
